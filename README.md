# Get-It-Done

A web-based todo list with focus on simplicity and productivity

## Features

- Add tasks
- Mark tasks as done
- Delete tasks
- Totally anonymous

## How to run

1. Clone this repository
2. Run `pnpm install`
3. Run `pnpm dev`

## Tech Stack

- React
- Vite
- TailwindCSS
- Radix UI
